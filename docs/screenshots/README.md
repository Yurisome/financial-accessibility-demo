# 스크린샷 저장 위치

데모 발표 및 제안서용 스크린샷을 이 폴더에 저장합니다.

## 저장 방법 (Android Studio)

1. Android Studio에서 앱 실행
2. 각 화면에서 **Logcat 옆 Camera 아이콘** 클릭 (또는 `Ctrl+S` / `Cmd+Shift+S`)
3. 아래 파일명으로 이 폴더에 저장

## 파일 목록

| 파일명 | 화면 | 내용 |
|--------|------|------|
| `01_home.png` | 홈 화면 | 앱 시작 화면, 버튼 2개 |
| `02_select.png` | 문서 선택 | 카드 명세서 / 신분증 선택 |
| `03_result_card.png` | 카드 명세서 결과 | AI 쉬운 설명 + 주요 정보 |
