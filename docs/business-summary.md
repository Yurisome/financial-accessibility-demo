# 금융 문서 도우미 - 사업 요약

## 서비스 개요

**서비스명:** 금융 문서 도우미

**한 줄 설명:**
금융 문서를 찍으면 AI가 쉬운 말로 설명해주는 서비스

**핵심 가치:**
단순히 글자를 읽어주는 것이 아니라, 문서의 의미를 쉬운 한국어로 풀어서 설명합니다.

---

## 문제 정의

금융 문서(카드 명세서, 청구서, 신분증 등)는 전문 용어와 작은 글씨로 가득합니다.
이를 혼자 이해하기 어려운 사람들이 실제로 많습니다.

**주요 타겟 사용자:**

| 그룹 | 어려움 |
|------|-------|
| 시각장애인 | 문서를 직접 읽을 수 없음 |
| 고령층 | 작은 글씨, 전문 용어 이해 어려움 |
| 디지털 취약계층 | 앱/인터넷 뱅킹 사용 불편 |
| 금융 문서 미숙 사용자 | 금융 용어에 익숙하지 않음 |

---

## 솔루션

1. **카메라로 문서 촬영** (현재: Mock 데이터로 대체)
2. **OCR로 텍스트 추출** (현재: Mock 데이터로 대체)
3. **AI가 내용을 쉬운 말로 변환**
4. **화면 표시 + 음성 안내**

### 변환 예시

| 원본 문서 | AI 설명 |
|----------|---------|
| 청구금액 1,230,000원 / 결제일 2026-05-25 | 이번 달 카드값은 123만원입니다. 5월 25일에 자동 결제될 예정입니다. |
| 921029-1\*\*\*\*\*\* | 주민등록번호 뒷자리는 안전하게 가렸습니다. |

---

## 현재 데모 상태

Android 네이티브 앱(Java) 기반 Mock 데모입니다.

**구현 완료:**
- 홈 화면 → 문서 선택 → 분석 결과 전체 흐름
- 카드 명세서 / 신분증 두 가지 Mock 분석 결과
- 접근성 지원: TalkBack 스크린 리더 대응, 큰 글씨 UI

**미구현 (향후 연동 예정):**
- 실제 카메라 촬영
- OCR 텍스트 추출
- AI 분석 API 연동
- 음성 안내 TTS

---

## 기술 구조 (목표)

```
Android 앱
    |
    | (사진 + 문서 종류)
    v
Spring Boot API
    |
    |-- OCR API (네이버 CLOVA / Google Vision)
    |-- AI 분석 (Claude API / GPT)
    v
쉬운 설명 + 주요 정보 반환
    |
    v
Android 화면 표시 + TTS 음성 안내
```

## 차별점

| 일반 OCR 앱 | 금융 문서 도우미 |
|------------|----------------|
| 글자만 읽어줌 | 의미를 풀어서 설명 |
| 전문 용어 그대로 출력 | 쉬운 한국어로 변환 |
| 시각장애인 대응 미흡 | TalkBack + 음성 안내 |
| 고령층 UI 고려 없음 | 큰 글씨, 큰 버튼, 고대비 |
